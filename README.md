# AdventureWorks Business Intelligence Dashboard

## Project Overview
This project simulates a real-world Business Intelligence Analyst role for AdventureWorks Cycles. The goal was to transform raw data into interactive dashboards and actionable business insights.

## Objectives
- Track key performance indicators (KPIs)
- Compare regional sales performance
- Analyze product-level trends
- Identify high-value customer segments

## Tools Used
- Power BI
- DAX
- Data Modeling
- Data Visualization

## Key Insights
- Identified top-performing regions based on revenue
- Highlighted high-margin product categories
- Analyzed customer purchasing behavior

- Power BI dashboard (.pbix)
